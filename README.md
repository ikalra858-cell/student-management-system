# Student Management System (Python)

A simple console-based Student Management System built using Python.  
This project demonstrates basic CRUD operations with file handling and dictionary data structures.

---

## 🚀 Features

- Add new student  
- View all students  
- Search student by name  
- Search student by ID  
- Update student details  
- Delete student records  
- Persistent storage using file handling (`students.txt`)  

---

## 🛠️ Tech Stack

- Python  
- File Handling  
- Dictionary Data Structure  
- Command Line Interface (CLI)
